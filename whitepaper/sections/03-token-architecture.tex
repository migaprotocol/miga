\subsection{Bonding Curve Mechanics}

The bonding curve provides fair price discovery:

\begin{itemize}
    \item \textbf{Start Price}: 0.0000001 SOL per token
    \item \textbf{End Price}: 0.00001 SOL per token (100x increase)
    \item \textbf{Curve}: Linear progression based on tokens sold
    \item \textbf{Slippage Protection}: Minimum tokens out parameter
\end{itemize}

\begin{equation}
\text{Price} = \text{StartPrice} + \frac{\text{TokensSold}}{\text{TotalSupply}} \times (\text{EndPrice} - \text{StartPrice})
\end{equation}

\subsection{Governance Power Formula}

\begin{equation}
\text{vePARS} = \min(\text{PARS}, \text{MIGA}) \times \sqrt{\frac{\text{lock\_duration}}{\text{max\_duration}}}
\end{equation}

This enforces:
\begin{itemize}
    \item \textbf{Balance}: Need both tokens---whales without MIGA cannot dominate
    \item \textbf{Commitment}: Longer locks earn more power
    \item \textbf{Diminishing Returns}: Square root prevents capture
\end{itemize}
