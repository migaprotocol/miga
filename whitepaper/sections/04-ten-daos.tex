\subsection{DAO Token Architecture (ERC-4626)}

Each DAO operates on EVM with:
\begin{itemize}
    \item \textbf{DaoVault}: ERC-4626 vault holding assets
    \item \textbf{DaoShare}: DAO token = shares in vault
    \item \textbf{BondDepository}: Discounted entry with vesting
\end{itemize}
