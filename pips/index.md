## Token Architecture

PARSDAO uses a **1 governance token + 10 DAO program token** model:

- **PARS**: Single governance token for voting, delegation, and protocol control
- **10 DAO Tokens**: Capital allocation instruments (ERC-4626 shares) with no governance power

See [PIP-8](PIPs/pip-0008-ten-dao-governance.md) for the full architecture.
